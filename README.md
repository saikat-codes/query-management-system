# Query Management System

A full-stack web app where users can submit queries and admins can manage them.

## Features
- User query submission
- Admin panel to view, update, delete and search queries
- Email notifications via Nodemailer
- Telegram notifications via Telegram Bot API

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB + Mongoose
- Notifications: Nodemailer, Telegram Bot API
